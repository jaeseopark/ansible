## RPI 32-bit distro:

Upgrade libseccomp2 for transmission to work correctly (ARMv7)

```
wget http://ftp.us.debian.org/debian/pool/main/libs/libseccomp/libseccomp2_2.5.3-2_armhf.deb
sudo dpkg -i libseccomp2_2.5.3-2_armhf.deb
```
